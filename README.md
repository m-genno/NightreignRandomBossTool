# Nightreign 3日目ボス候補ツール (Android)

ELDEN RING NIGHTREIGN で、3日目のボス（夜の王）が分からないときに、
**1日目・2日目の夜ボスから候補を絞り込む**ためのAndroidアプリです。

## 機能

- 1日目の夜ボスを選ぶと、3日目のボス候補が表示される
- 2日目の夜ボスを選ぶと、3日目のボス候補が表示される
- 1日目・2日目の両方を選ぶと、候補が積集合で絞り込まれ精度が上がる
- 夜ボスはリストから選択。**一部入力でリアルタイムに絞り込み**
  - 表示名（漢字）と **よみがな** の両方にマッチ
  - 「鈴」でも、変換前の「す」でも、カタカナ「ス」でも `鈴玉狩り` が出る
- **リセット**ボタンで入力をクリア
- ボス情報は**コードに直書きせず外部テキストファイル**で管理

## ビルド / 実行

1. このフォルダを **Android Studio** で開く（`File > Open`）
2. Gradle Sync が走る（初回は Gradle 8.7 を自動ダウンロード）
3. 実機またはエミュレータで Run ▶

> コマンドラインでビルドする場合は、Gradle 8.7 以上で
> `gradle wrapper` を一度実行して wrapper を生成してから
> `./gradlew assembleDebug` を実行してください。
> （Android Studio で開けば wrapper は自動生成されます）

## ボスデータの編集方法

データは2つのテキストファイルで管理します。**コードを触る必要はありません。**

### ファイルの場所

- **編集用の元データ（リポジトリ内）**: `app/src/main/assets/`
  - `bosses.txt` … ボスの「よみがな」辞書（検索用）
  - `nightlords.txt` … 夜の王ごとに出やすい夜ボスの対応表
  - ここを編集して再ビルドすると初期データになります。

- **端末上で編集する場合**: 初回起動時に上記が
  `Android/data/com.nightreign.bosstool/files/` にコピーされます。
  ファイルマネージャ等で編集 → アプリ右上の **再読み込み**（↻）で反映。
  元に戻したいときは メニュー（⋮）→ **「サンプルデータに戻す」**。

### `bosses.txt`（よみがな辞書）

```
表示名, よみがな
鈴玉狩り, すずたまがり
```

- よみがなを書いておくと、「す」などのあいまい入力で検索できます。
- よみがな省略時は表示名がそのまま読みになります。

### `nightlords.txt`（対応表）

```
3日目ボス | 1日目に出やすい夜ボス | 2日目に出やすい夜ボス
三面獣グラディウス | 鈴玉狩り;夜の騎兵 | 黄金樹の化身;坩堝の騎士
```

- 夜ボスが複数のときは `;` で区切ります。
- 片方しか分からない場合は、その項目を空にしても構いません。
- 夜ボスの選択リストは、このファイルに登場する夜ボスから自動生成されます。

> ⚠️ 同梱データは **動作確認用のサンプル**です。
> 実際のゲームの対応に合わせて `nightlords.txt` を編集してください。
