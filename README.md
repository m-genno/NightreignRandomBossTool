# Nightreign 3日目ボス候補ツール (Android)

ELDEN RING NIGHTREIGN で、3日目のボス（夜の王）が分からないときに、
**1日目・2日目の夜ボスから候補を絞り込む**ためのAndroidアプリです。

## 機能

- 1日目の夜ボスを選ぶと、3日目のボス候補が表示される
- 2日目の夜ボスを選ぶと、3日目のボス候補が表示される
- 1日目・2日目の両方を選ぶと、候補が積集合で絞り込まれ精度が上がる
- 夜ボスはリストから選択。**一部入力でリアルタイムに絞り込み**
  - 表示名（漢字）と **よみがな** の両方にマッチ
  - 「鈴」でも、変換前の「す」でも、カタカナ「ス」でも `鈴玉狩り` が出る
- **リセット**ボタンで入力をクリア
- ボス情報は**コードに直書きせず外部テキストファイル**で管理

## ビルド / 実行

1. このフォルダを **Android Studio** で開く（`File > Open`）
2. Gradle Sync が走る（初回は Gradle 8.7 を自動ダウンロード）
3. 実機またはエミュレータで Run ▶

> コマンドラインでビルドする場合は、Gradle 8.7 以上で
> `gradle wrapper` を一度実行して wrapper を生成してから
> `./gradlew assembleDebug` を実行してください。
> （Android Studio で開けば wrapper は自動生成されます）

## ボスデータの編集方法

データは2つのテキストファイルで管理します。**コードを触る必要はありません。**

### データの考え方

夜の王ごとに「1夜（1日目）に出る夜ボス」「2夜（2日目）に出る夜ボス」を持ちます。

- 1日目の夜ボスを選ぶ → それを **1夜** に持つ夜の王が候補になる
- 2日目の夜ボスを選ぶ → それを **2夜** に持つ夜の王が候補になる
- 両方を選ぶ → 両方を満たす夜の王だけに絞り込まれる（**積集合**・精度UP）

1夜と2夜では出現する夜ボスが異なるため、1日目用と2日目用で選択リストも別になります。

### ファイルの場所

- **編集用の元データ（リポジトリ内）**: `app/src/main/assets/`
  - `bosses.txt` … 夜ボスの「よみがな」辞書（検索用）
  - `nightlords.txt` … 夜の王ごとの 1夜/2夜 出現ボス
  - ここを編集して再ビルドすると初期データになります。

- **端末上で編集する場合**: 初回起動時に上記が
  `Android/data/com.nightreign.bosstool/files/` にコピーされます。
  ファイルマネージャ等で編集 → アプリ右上の **再読み込み**（↻）で反映。
  元に戻したいときは メニュー（⋮）→ **「初期データに戻す」**。

### `bosses.txt`（よみがな辞書）

```
表示名, よみがな
鈴玉狩り, すずたまがり
```

- よみがなを書いておくと、「す」などのあいまい入力で検索できます。
- よみがな省略時は表示名がそのまま読みになります。

### `nightlords.txt`（夜の王ごとの出現ボス）

```
夜の王 | 1夜（1日目）の夜ボス | 2夜（2日目）の夜ボス
グラディウス | 亜人の女王＋亜人の剣聖;鈴玉狩り | 忌み鬼;ツリーガード
ナメレス | * | *
```

- 夜ボスが複数のときは `;` で区切ります。
- `*`（または「全種類」）はどの夜ボスにもマッチします（ナメレス用）。
- `*;-鈴玉狩り` のように `-ボス名` を足すと、全種類から特定ボスを**除外**できます
  （例: ナメレスは全種類だが、追加ボスだけは候補に含めない）。
- 選択リストは、この表の各列に登場する夜ボスから自動生成されます。
- 通常版/常夜版は同じ夜の王として統合しています。

> データ出典: [JBJBGAME](https://jbjbgame.com/post-28051/)

## Web版（フロントエンドのみ）

`docs/` に、同じ仕様の静的Webアプリ（HTML/CSS/JavaScript）が入っています。
サーバー不要で、ブラウザだけで動きます。

- `docs/index.html` … 画面
- `docs/app.js` … ロジック（読み検索・絞り込み）
- `docs/style.css` … スタイル
- `docs/data/bosses.txt`, `docs/data/nightlords.txt` … データ（形式はAndroid版と同じ）

> データはAndroid版（`app/src/main/assets/`）と同じ内容のコピーです。
> 更新時は両方を合わせてください。

### ローカルで試す

`file://` で直接開くと `fetch` がブロックされるため、簡易サーバー経由で開きます。

```
cd docs
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

### 無料で公開する（GitHub Pages）

このリポジトリだけで無料公開できます。**1回だけ**以下を設定してください。

1. GitHub のリポジトリ → **Settings** → 左メニュー **Pages**
2. **Build and deployment** → **Source** = `Deploy from a branch`
3. **Branch** = `main`、フォルダ = `/docs` を選び **Save**
4. 数十秒〜数分後、ページ上部に公開URLが出ます
   （このリポジトリなら `https://m-genno.github.io/NightreignRandomBossTool/`）

以降は `git push` するたびに自動で再公開されます。
